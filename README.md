# ₿ Bitcoin Price Prediction using Machine Learning

A full-stack web application that leverages Machine Learning to forecast Bitcoin price trends. It features a robust Python FastAPI backend for data processing and model training, and a stunning, responsive frontend built with modern web technologies (HTML, CSS, JS) and Chart.js.

## 🌟 Features

- **Real-Time Data Processing**: Fetches and processes the latest historical Bitcoin price data.
- **Machine Learning Model**: Analyzes historical trends, technical indicators, and price movement patterns to predict the next 24-hour forecast.
- **Premium UI/UX**: A beautifully designed "Glassmorphism" interface with dynamic mesh gradients, smooth micro-animations, and a responsive layout.
- **Interactive Data Visualization**: Interactive historical and predictive price charts powered by Chart.js.
- **Full-Stack Integration**: The FastAPI backend acts as a unified server, smoothly serving both the RESTful prediction endpoints and the static frontend UI.

## 🚀 How to Run the Project

The project is cleverly configured so that the FastAPI backend automatically serves the frontend interface. **You only need to start the backend server!**

### Prerequisites
- Python 3.8 or higher installed on your machine.

### Step-by-Step Guide

1. **Open Your Terminal**
   Navigate to the `backend` folder of the project:
   ```bash
   cd "d:\Bitcoin Price Prediction Using Machine Learning\backend"
   ```

2. **Activate the Virtual Environment**
   Activate the existing virtual environment to ensure dependencies don't conflict with your global Python installation:
   - **Windows:** 
     ```powershell
     .\venv\Scripts\activate
     ```
   - **Mac/Linux:** 
     ```bash
     source venv/bin/activate
     ```

3. **Install Dependencies**
   Make sure all the required Python packages are installed:
   ```bash
   pip install -r requirements.txt
   ```

4. **Start the Application Server**
   Start the FastAPI application using `uvicorn`:
   ```bash
   uvicorn app.main:app --reload
   ```

5. **View the Dashboard**
   Open your favorite web browser and navigate to:
   **[http://localhost:8000](http://localhost:8000)**

   *Note: You can view the interactive API documentation automatically generated by FastAPI at [http://localhost:8000/docs](http://localhost:8000/docs).*
